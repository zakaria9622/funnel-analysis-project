# Dataset

## Why the full dataset is not included

The full event-level dataset used for the final analysis is not included in this repository because of file size limitations.

## How to use the full dataset

To run the project with the full dataset, place the file here:

`data/ecommerce_events.csv`

## Sample dataset

A small sample file is included to make the Python pipeline executable and reviewable.
